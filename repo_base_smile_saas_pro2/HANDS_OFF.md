# Guía Operativa (Hands-off) - Smile SaaS PRO 2

¡Bienvenido al futuro de tu consultorio odontológico! Esta guía te ayudará a navegar el sistema y entender el flujo comercial diseñado para maximizar tus cierres.

## 🚀 El Ciclo del Éxito Comercial

Nuestro sistema no es solo una ficha médica, es un motor de ventas. Sigue este flujo para cada paciente:

1.  **Captación (Paciente)**: Registra los datos básicos. Un paciente bien documentado es un paciente que confía.
2.  **Expediente (Caso)**: Crea un caso por cada motivo de consulta. Recuerda cargar siempre las fotos frontales para el diseño.
3.  **Efecto "WOW" (Diseño)**: Utiliza el editor de sonrisa para mostrarle al paciente su futuro antes de empezar. La visualización aumenta la tasa de aceptación en un 40%.
4.  **Propuesta (Presupuesto)**: Ofrece siempre **3 alternativas** (Essential, Premium, Pro). Al dar opciones, el paciente deja de preguntarse "¿Lo hago o no?" para preguntarse "¿Cuál de estos me conviene?".
5.  **Persistencia (CRM/Seguimiento)**: Si no aceptan en el momento, registra cada contacto en el Timeline. El 80% de las ventas ocurren después del quinto contacto.

## 🔐 Seguridad y roles

- **Administrador**: Acceso total a métricas comerciales y facturación estimada.
- **Odontólogo**: Enfoque clínico y diseño de sonrisa.
- **Ventas/Asistente**: Gestión de seguimiento y carga de fotos.

## 🛠️ Mantenimiento Diario

- **Dashboard**: Revisa los indicadores cada mañana. Los "Presupuestos en Seguimiento" son tu prioridad nº1.
- **Cierre de Casos**: Cuando un tratamiento termine, cámbialo a estado "Cerrado" para mantener tus métricas de dashboard limpias y enfocadas.

Quiero construir el sistema visual premium para un SaaS odontológico enfocado en estética dental y gestión comercial.

Objetivo:
crear una base de UI reutilizable, moderna, clínica y premium.

Necesito:
- paleta de colores
- tipografía
- spacing system
- componentes base: botones, inputs, cards, badges, tablas
- layout base: sidebar, topbar, contenedor
- tokens reutilizables
- ejemplos de uso

Restricciones:
- React + TypeScript + Tailwind
- nombres en español
- estética premium (no admin genérico)
- componentes reutilizables
- mucho espacio en blanco
- evitar saturación visual

No quiero:
- pantallas completas todavía
- lógica de negocio
- conexión a backend

Entrega:
- componentes base listos
- estructura clara
- código limpio y modular

Al finalizar:
ejecutar npm run build y npm run lint
reportar errores y correcciones

 Quiero construir el mockup premium del módulo de presupuesto y seguimiento comercial para un SaaS odontológico enfocado en diseño de sonrisa y cierre comercial.

Objetivo:
crear una pantalla clara, premium y accionable para presentar presupuesto, visualizar opciones de tratamiento y ordenar el seguimiento comercial.

Necesito:
- encabezado del presupuesto
- resumen económico
- listado de opciones de tratamiento
- una opción recomendada visualmente destacada
- estado de negociación
- próxima acción comercial
- timeline de seguimiento
- observaciones comerciales

Restricciones:
- React + TypeScript + Tailwind
- nombres en español
- usar datos mock centralizados en archivos separados
- no conectar backend
- no meter lógica contable o fiscal real todavía
- mantener una UI premium, moderna, clínica y orientada a cierre
- evitar saturación visual
- dejar la arquitectura lista para integración futura

Quiero:
- componentes reutilizables
- pantalla montada dentro del shell existente
- layout jerárquico y prolijo
- código modular, tipado y consistente

Al finalizar:
ejecutar npm run build y npm run lint
reportar errores encontrados y correcciones aplicadas
